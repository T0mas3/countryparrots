Country Parrots
==============


## Install
```
npm install
npm install imageoptim-cli -g
```

## generate parrots
```
npm run-script generate-parrots
```


## optimize gifs
```
npm run-script optimize-gifs
```

## generate zip
```
npm run-script generate-zip
```

## build index.html
```
npm run-script generate-index
```
